# Analisis-dataset-h-gado
Análisis de perfiles hepáticos en una muestra de 582 pacientes

Proyecto: Análisis Bioquímico de Pacientes Hepáticos
Este proyecto analiza un dataset de pacientes para identificar marcadores clave en el diagnóstico de enfermedades del hígado.

🔍 Hallazgos Clave:
Correlaciones: Se identificó una fuerte relación entre TB y DB (0.87) y Sgot y Sgpt (0.79).

Variables Predictoras: Las bilirrubinas resultaron ser los indicadores más fiables de enfermedad.

Valores Atípicos: Se detectó una alta presencia de outliers en las enzimas (Sgot), reflejando casos clínicos agudos.
